# News

A  beautiful responsive news website using next js

## Demo

https://news-sh.netlify.app/
## Features

- Smooth Animation
- Nice looking UI
- SEO friendly
- Sign In
- Sign Up
- Api Intrigration

## Installation


Install the dependencies and run the development server

```bash
npm install
npm run dev
# or
yarn
yarn dev
```

